# Lang Academy

Math Academy-style Chinese learning app: DAG knowledge graph, mastery gating, and spaced repetition.

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4

## Getting Started

```bash
npm install
npm run dev
```

## Project Structure

```
src/
  components/    # UI components
  data/          # Knowledge graph data
  engine/        # SRS + mastery logic
  types/         # TypeScript interfaces
  App.tsx
  main.tsx
```
